# Sabbatical 2026 — Itinerary

**Aug 8 – Sep 7, 2026.** Disc golf road trip out of Colorado, in a camper.

> **Canonical schedule.** Transferred from the Apple Notes "Sabbatical 2026"
> note (Trips folder) — this file is now the source of truth. The Apple Note can
> be archived. The **KC → Detroit disc golf leg (Aug 17–25)** is mapped in
> `../road-trip-map/` and its courses come from `../data/course-rankings/`.

## Shape of the trip

```
KC (start)  →  fly to Denver  →  Buena Vista (truck prep)
   →  disc east across CO  →  back to KC (base, ~Aug 13–16)
   →  ★ KC → Detroit disc golf leg (Aug 17–25) ★
   →  Worlds, spectating near Detroit (Aug 26–30)
   →  Detroit → Iowa City → Nebraska → BV/Denver → fly KC (Aug 31 – Sep 4)
   →  Labor Day weekend in KC (Sep 5–7)
```
